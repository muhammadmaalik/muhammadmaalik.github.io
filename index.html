Rocket Systems

TVC (Thrust Vector Control)

Thrust Vector Control is simply using a mechanism that can control your rocket's flight trajectory. It is up to you to make it complex or simple. Using just a system of servos and a micro-controller can help you steer your rocket for the time it is in the air. To make it more advanced you can add another Servo on the top of the rocket to make a lever for the parachute-which can save it from big heights.

Micro Controller

The Micro Controller I would use for this project would be an Arduino UNO because it's a light job for a microcontroller to control gyroscope sensors and to keep the rocket in line. I would simply feed it with C++ code, the Gyroscope sensor would be connected to digital pins on Arduino. I would use a total of 3 Servos, 2 for the TVC and one for the parachute control. Parachute is important because it's going to save my rocket and the computers inside it and I would use it again and again to perfect my code.
